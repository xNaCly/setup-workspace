### How to:

`main.py [type of workspace] [File path]`

- type of [workspace](https://github.com/xNaCly/setup-workspace/blob/master/src/config.py)
- file path: `[".", "./", "C:/dir"]`

```
example:
main.py --node C:\Users\User\Desktop
```
